MVF 1: Search

Description

The search engine will allow user to search for specific recipes, this is done through a search bar. Users will also be able to use a filter to help find more specific recipes. To accomplish this a database is used to store all the recipes and to return result based on what the user entered. The search engine is global meaning that it will appear on multiple pages.

Resources and tools

HTML5:
HTML5 is a markup languages used for web development. It is responsible providing a structure and presenting contents on the web. We're using html5 as it's one of the most common languages for web developement and we all have some experience using it, it is also free. there isn't really an alternative to html as majority of browser used html.

HTML5 references: https://developer.mozilla.org/en-US/docs/Web/HTML/Element

CSS3:
CSS is a stylesheet language thats compatible with html5. It is used to deal with presentation of a web page, essentially it is used to make a web page look pretty. This is going to be use for our web design as there isn't really an alternative for CSS.

CSS references: https://developer.mozilla.org/en-US/docs/Web/CSS/Reference

PHP:
PHP is a server scripting language for web development. This is going to be used to handle account management in our project such as registration and login. Alternatives are Javascript/Node.js.

PHP website: https://www.php.net/ Node.js website: https://nodejs.org/en/

mySQL:
It's a database management Systems. We're going to use it to store account informartion.

Collaborative Workspace

Google docs:
All team members are able to access a shared google doc to edit Assignment 1. A link to join is accessible through teams.

Trello:
Trello allows for different tasks to easily be managed and accessed by group members on cards. Members can add and edit tasks and communicate through comments. Trello board has been implemented on teams so all team members can view the board through teams without needing to view an external app.

GitHub:
GitHub allows for file sharing and version management between members. Every member is able to upload files to GitHub to be previewed or downloaded. A link to access GitHub is available to all group members on teams.
